# AI Rune (Upcoming Feature)

One of the biggest future goals of this project is building an AI assistant called:

## ✨ AI Rune

AI Rune is planned as a specialized Ragnarok Mobile mechanics assistant.

Instead of being a generic chatbot, the AI will understand:

- Internal Lua formulas
- Damage calculations
- Buff interactions
- Hidden mechanics
- Card effects
- Monster behaviors
- Skill systems
- Equipment interactions

The vision is to create an intelligent archive assistant capable of helping players and researchers better understand how the game actually works internally.

---

# Planned AI Flow

The AI system will likely use a Retrieval-Augmented Generation (RAG) architecture.

High-level flow:

```txt
User Question
      ↓
Search relevant ROM archive data
      ↓
Inject formulas / mechanics into context
      ↓
AI-generated explanation
```

Example future questions:

1. "Why does Crit Rate become 0 during Hallucination Walk?"
2. "Which formula controls final damage reduction?"
3. "Why doesn't this card affect MVP monsters?"
4. "How does this buff stack internally?"

The AI will answer based on the actual archived formulas and preserved game data
